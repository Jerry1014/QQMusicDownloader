# QQMusicDownloader
网络编程技术课设，通过QQ音乐的接口，提供歌曲的试听和下载

## 功能
- 搜索试听下载歌曲，曲库依据选择的api决定

## 特色
- 可下载版权歌曲（存在bug，暂时下线）
- 采用非转基因原料，纯正无添加剂(*废话，你会往课设里添广告?*)
- 网页右下角有可爱live2d小人(*程序员可喜欢可爱的妹子了*)
